# Gesture Braille Keyboard Support

Gesture Braille Keyboard is an accessible keyboard app that supports both gesture typing and Braille input.

## Setup

1. Install the app on your iPhone.
2. Open **Settings**.
3. Go to **General > Keyboard > Keyboards > Add New Keyboard**.
4. Select **Gesture Braille Keyboard**.
5. Enable **Allow Full Access** if needed for supported features (e.g. voice feedback).

## Features

- Gesture keyboard with swipe and tap input
- Braille keyboard with swipe controls
- Smart suggestions
- Smart deletion
- Switch between gesture and Braille input modes

## Troubleshooting

### The keyboard does not appear
Make sure the keyboard has been added in **Settings > General > Keyboard > Keyboards**.

### Some features do not work
Check whether **Allow Full Access** is enabled if the feature requires it.

### I want to report a bug
Please email the issue details, device model, and iOS version.

# XXX Keyboard Support

XXX Keyboard is an accessible iPhone keyboard designed for blind and low-vision users. It supports both QWERTY gesture typing and Braille input, with voice feedback for supported interactions.

## Setup

1. Install the app on your iPhone.
2. Open **Settings**.
3. Go to **General > Keyboard > Keyboards > Add New Keyboard**.
4. Select **XXX Keyboard**.
5. Enable **Allow Full Access** if needed for supported features such as voice feedback.

## Features

- Standard QWERTY gesture keyboard with swipe and tap input
- Visible gesture path and magnified key feedback
- Smart word suggestions while typing
- Braille keyboard with touch-based input and gesture controls
- Braille auto-correction for improved accuracy
- Voice feedback for supported interactions
- Easy switching between gesture and Braille input modes

## Braille Controls

- **Swipe left or right** to change word candidates
- **Swipe down** to commit a word or insert a space
- **Swipe up** to delete letters, spaces, or committed words
- **Swipe down with three fingers** to switch keyboards

## Troubleshooting

### The keyboard does not appear
Make sure the keyboard has been added in **Settings > General > Keyboard > Keyboards**.

### Some features do not work
Check whether **Allow Full Access** is enabled if the feature requires it.

### I want to report a bug
Please include the issue details, device model, and iOS version in your email.
